# Monica's Beach Bach Party

Codex/local-preview ready static website.

## Run locally

```bash
cd ~/Desktop/monicas-beach-bach-party
python3 -m http.server 8000
```

Open: http://localhost:8000

## Image folder

All image paths in `index.html` use `./art/filename`, so keep the `art` folder next to `index.html`.

## Cover photo

The cover has a local-only file picker. Select a photo in the browser to preview it behind the masthead. It does not upload anywhere.
